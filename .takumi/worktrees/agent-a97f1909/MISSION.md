# SemHex — Mission

> "inventează aceeași limbă ca și culorile"
> "invent the same language as colors"

**Find the dimensions of meaning, then make every meaning a coordinate in that space.**

Colors have 3 dimensions: Red, Green, Blue.
6 characters encode any color a human can see: #FF0000

Meaning has its dimensions too.
We find them. We encode every meaning as coordinates.
Short code. No dictionary. The code IS the meaning.

```
#FF0000 = red
$4B.C4  = frustrated with a bug
```

That's it. That's the whole mission.
